🍣 Contador de Sushi

Aplicação web simples para contabilizar a quantidade de peças consumidas em um rodízio de sushi.

Ideal para usar com amigos e acompanhar quem está “ganhando” no rodízio 😄

🚀 Funcionalidades
➕ Adicionar peças consumidas
➖ Remover peças
📊 Exibir total consumido
💾 Persistência automática no navegador (localStorage)
🔄 Resetar contagem
🛠️ Tecnologias utilizadas
React
Vite
JavaScript (ES6+)
HTML + CSS
